# RL-GUIDED-MULTI-HOP-LEGAL-QUESTION-ANSWERING-SYSTEM
Combining NLP Pipelines with Reinforcement Learning for Adaptive Reasoning over Legal Documents.



⚠️ IMPORTANT INSTALL METHOD

Because of Torch CPU index, install like this:

pip install -r requirements.txt

If Torch fails:

pip install torch==2.2.2+cpu --index-url https://download.pytorch.org/whl/cpu
pip install -r requirements.txt --no-deps
